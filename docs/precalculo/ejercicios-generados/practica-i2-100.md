## gen-i2-001
- Tema: Funciones / Dominio raíz/racional
- Patrón: Dominio raíz/racional
- Enunciado: Determine el dominio de f(x)=sqrt(x+2)/(x-3).
- Respuesta: [-2,3) union (3,infty)

## gen-i2-002
- Tema: Funciones / Transformaciones
- Patrón: Transformaciones ordenadas de funciones
- Enunciado: A f(x)=x^2 se le aplica: derecha 2, reflexión eje Y, bajar 3. Determine fórmula resultante.
- Respuesta: y=x^2+4x+1

## gen-i2-003
- Tema: Funciones / Operaciones de funciones
- Patrón: Composición y operaciones de funciones
- Enunciado: Sean f(x)=1/(x-2) y g(x)=(x+1)/(x-3). Determine restricciones principales del dominio de f/g.
- Respuesta: x != 2,3,-1

## gen-i2-004
- Tema: Funciones / Inversa racional
- Patrón: Función inversa racional o por tramos
- Enunciado: Considere f(x)=(2x+1)/(x-3). Determine f^{-1}(x).
- Respuesta: (3x+1)/(x-2)
